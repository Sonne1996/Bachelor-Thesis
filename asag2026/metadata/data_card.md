## 4. Intended Use
The dataset is intended for:
- Research in automatic grading
- Benchmarking LLM-based evaluation systems
- Bias and robustness analysis

## 5. Limitations
- Contains model-generated components
- Possible grading bias across review models
- Language sensitivity effects possible
- Not validated against human gold-standard grading
